\doxysubsection{Detailed Description}
The \doxylink{classObstacle}{Obstacle} class is a class Derived from \doxylink{classPositionable}{Positionable} class. 

The \doxylink{classObstacle}{Obstacle} class is used to represent the obstacles in the game. The \doxylink{classObstacle}{Obstacle} is non movable object that blocks the way of the human and the monster.
