\doxysubsection{Detailed Description}
The \doxylink{classCustomGraphicsItem}{Custom\+Graphics\+Item} class representing objects on the game map. 

The \doxylink{classCustomGraphicsItem}{Custom\+Graphics\+Item} class derives from QGraphics\+Pixmap\+Item. Its main responsibility is to render the object with the given asset on a given position.
